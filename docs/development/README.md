# Development Guides

<doc_index>
This directory contains practical workflow guides for contributors.
</doc_index>

## Guides

- `github-workflow.md` - issue-first GitHub workflow, branches, pull requests, and review.
- `git-github-cli-workflow-tutorial.md` - beginner-oriented command walkthrough for the issue-first workflow using Git and the GitHub CLI.
- `conventional-commits.md` - commit message and pull request title standards.
- `automation.md` - local hooks and GitHub Actions that support the contribution workflow.

## Maintenance

Update these guides when repository workflow, automation, branch protection, or release policy changes.
